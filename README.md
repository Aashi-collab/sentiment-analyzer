Objective

The main objective of this project is to build a model that can automatically understand human emotions from text and classify them into different sentiment categories.

Features
Input text box for user to enter custom sentences
Real-time sentiment prediction
Text preprocessing (cleaning, tokenization, vectorization)
Classification using Machine Learning model
Simple and interactive UI using Streamlit


NLP & Machine Learning Approach

The project follows an end-to-end NLP pipeline:

🔹 Text Preprocessing
Lowercasing
Removing punctuation & stopwords
Tokenization
Stemming / Lemmatization


🔹 Feature Extraction
TF-IDF (Term Frequency - Inverse Document Frequency)
or Count Vectorization


🔹 Model Used
Logistic Regression / Naive Bayes (depending on your model)
Model Evaluation

The model performance is evaluated using:

Accuracy score
Confusion matrix
Precision, Recall, F1-score

These metrics ensure the model is performing well on unseen data.

Tech Stack
Programming Language: Python
Libraries: Pandas, NumPy, Scikit-learn, NLTK / SpaCy
NLP Tools: TF-IDF / CountVectorizer
Deployment: Streamlit

Tools: Jupyter Notebook, VS Code


🌐 Deployment

The application is deployed using Streamlit, allowing users to input text and instantly get sentiment predictions.
